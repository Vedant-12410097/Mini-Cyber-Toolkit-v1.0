## ⚙️ Tool Descriptions and Tips

### IP Information

Shows your current IPv4 and IPv6 addresses. Useful to verify your network setup. It pulls network info from your system and online services.

### Port Scanning

Finds open ports on a target device on the local network or internet. Helps check whether certain services are available or firewall rules apply.

- Target can be a hostname or an IP address.  
- The scan uses TCP protocol.  

### Password Generation

Creates random passwords with options to include letters, numbers, and symbols.

- Adjust length for weak or strong passwords.  
- Copy the password with one click for easy use.

### Hash Generation & Verification

Calculate hashes like MD5 or SHA256 for files or text input. Also check if a file was changed by comparing hashes.

- Works for integrity checks.  
- Supports drag and drop for files.

### Network Traffic Statistics

Shows a summary of your device’s network activity in real time. Helps monitor the amount of data used by apps.

### System Information & Monitoring

Displays CPU use percentage, memory available vs used, and storage disk space.
